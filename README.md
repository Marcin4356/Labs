# Labs

A collection of Python laboratory exercises and experiments for data science, machine learning, and computational problem-solving.

## 📋 Description

This repository contains practical lab exercises for learning and experimenting with Python, data science, and machine learning concepts. Each lab includes explanations, code examples, and exercises.

## 🛠️ Tech Stack

- **Language:** Python 3.8+
- **Data Science:** Pandas, NumPy
- **Machine Learning:** Scikit-learn, TensorFlow, PyTorch
- **Visualization:** Matplotlib, Seaborn
- **Notebooks:** Jupyter Notebook

## 📋 Prerequisites

```bash
# Python 3.8 or higher
# pip package manager
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Marcin4356/Labs.git
cd Labs
```

### 2. Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run Labs

```bash
# Using Jupyter Notebook
jupyter notebook

# Or run Python scripts directly
python lab1_basics.py
```
